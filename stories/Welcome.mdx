# Useresumate Design System

Component library for the Useresumate AI resume optimization platform.

## Stack

- **UI Primitives**: shadcn/ui (new-york style) with Radix UI
- **Theme**: OKLCH color tokens, light/dark mode via CSS class toggle
- **Icons**: lucide-react
- **Typography**: Geist Sans (body), Geist Mono (code), Space Grotesk (headings), Lora (blog)

## Usage

Use the **theme switcher** in the toolbar above to toggle between light and dark modes.

Stories are co-located next to their components in `components/ui/`. To add a new story, create a `*.stories.tsx` file alongside the component.
